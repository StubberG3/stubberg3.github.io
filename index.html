  • Front-end: React.js and Tailwind CSS for interactive and responsive UI/UX.
  • Back-end: Django for robust data management and API services.
  • Database: PostgreSQL for storing paper details, user data, and admin approvals.
  • AI Service: OpenAI for analyzing and verifying the relevance of the papers to mental health.
  • File Storage: AWS S3 for storing uploaded PhD papers
